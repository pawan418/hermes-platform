# Hermes Module System

Hermes features an extensible design supporting registration of module packages:

## Supported Modules
- **CRM**: Customer Relationships Manager
- **Sales**: Sales Pipeline
- **Academy**: LSPL Training modules
- **Voice**: Call and text engines
- **WhatsApp**: Webhook channels
- **HR**: Team directory
- **Finance**: Bills and receipts tracker
- **Support**: Ticketing system
- **Knowledge**: Knowledge assets database
- **Analytics**: Statistics dashboard

## Commands
- `hermes module list`
- `hermes module enable <mod>`
- `hermes module disable <mod>`
- `hermes module install <mod>`
